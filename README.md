# safein Project (Frontend)
Safein's frontend
## Technology Stack 🖥
We used: 
- Angular CLI
- AWS Amplify
- NPM
- TypeScript

## Deploy 🛠
Safein project use railway for deploying database and API, developed using Spring and Spring Security with Maven file structure and database debeloped using MySQL.
